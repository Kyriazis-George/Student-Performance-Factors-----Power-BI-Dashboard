Student Performance Analysis Dashboard
Overview
This project analyzes student performance using a dataset containing academic results, behavioural factors, and demographic information. The goal is to identify key factors influencing student achievement and present insights through an interactive dashboard.
The dashboard was developed in Power BI using DAX measures and visual analytics to support data-driven decision making in an educational context.

Objectives


Analyse overall student performance in final exams


Compare performance across demographic groups


Evaluate the impact of behavioural factors on academic results


Identify key drivers of student success



Tools and Technologies


Power BI


DAX (Data Analysis Expressions)


Excel (CSV dataset)



Dataset Description
The dataset includes the following variables:


Exam scores


Attendance rate


Hours studied


Sleep duration


Gender


Motivation level


Family income


Learning disability status



Dashboard Structure
General Overview
Provides a summary of key metrics:


Average exam score


Attendance rate


Average study hours


Average sleep hours


Disability percentage


Total number of students


Includes KPI indicators for overall performance monitoring.
Performance Analysis
Focuses on academic outcomes:


Distribution of students by result rank (Upper Level, Merit, Pass, Fail)


Overall score distribution


Behaviour Analysis
Examines behavioural influence on performance:


Relationship between study hours and exam score


Relationship between sleep and exam score


Impact of attendance on performance


Demographics Analysis
Explores performance differences across groups:


Gender-based performance comparison


Impact of motivation level on exam scores


Family income influence on performance


Effect of learning disability on results



Key Insights


Motivation level is strongly correlated with higher academic performance


Increased study hours and consistent attendance are associated with better exam results


Socioeconomic factors such as family income may influence student outcomes


Behavioural factors have a significant impact on overall performance



Conclusion
This project demonstrates how behavioural and demographic variables can be used to explain variations in student performance. The dashboard provides clear, interactive insights that support analysis of academic outcomes and underlying influencing factors.
